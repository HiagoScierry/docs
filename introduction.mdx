## Base URL

Todos os endpoints usam a mesma base:

```
https://api.cordialy.ai/integrations/v1
```

## Autenticação

Todas as requisições precisam do header `X-API-Key`:

```bash
curl https://api.cordialy.ai/integrations/v1/leads \
  -H "X-API-Key: sua_chave_aqui"
```

Gere sua chave em **Plataforma → Integrações → API Keys**.

A API pública está disponível **apenas para planos Premium**. Planos Low Ticket não possuem acesso à API.

| Recurso | Starter | Growth | Enterprise | Enterprise Plus |

| Leads (leitura e escrita) | ✅ | ✅ | ✅ | ✅ |

| Follow-ups | — | ✅ | ✅ | ✅ |
| Base de conhecimento | — | ✅ | ✅ | ✅ |
| Agentes de IA | — | — | ✅ | ✅ |
| Relatórios | — | — | ✅ | ✅ |
| Webhooks (eventos de saída) | ✅ (2) | ✅ (5) | ilimitado | ilimitado |
| API Keys permitidas | 1 | 3 | ilimitado | ilimitado |

## Limites de taxa

| Plano | Req/s |
|---|---|
| Starter | 5 |
| Growth | 20 |
| Enterprise | 60 |
| Enterprise Plus | 150 |

Quando o limite é atingido, a API retorna `429 Too Many Requests`. O header `Retry-After` indica em quantos segundos você pode tentar novamente.
